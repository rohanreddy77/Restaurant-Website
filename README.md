# 🍽️ Foodie's Restaurant Website

A modern and responsive restaurant landing page developed using HTML and CSS.

## Features

- Responsive Design
- Modern Navigation Bar
- Hero Section
- Menu Cards
- About Section
- Contact Section
- Hover Effects
- Mobile Friendly

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

## Project Preview

This project demonstrates a clean and user-friendly restaurant website suitable for beginners learning front-end web development.
